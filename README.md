# Persistent Artificial Graph-based Neural Networks (PAGNNs)

## Environment Setup:

1. Clone this repository.
2. `cd path/to/cloned/PAGNNs/`
3. `python -m venv env`
4. `source env/bin/activate`
5. `pip install -e .`
